# 💰 Expense Tracker

A personal expense tracking web application built with **Python Flask**, **SQLite**, and **Chart.js** as part of my BCA Python Full Stack internship project.

---

## Features

- User Registration and Login with password hashing
- Add, Edit, and Delete expenses
- Categorize expenses (Food, Transport, Shopping, etc.)
- Set a monthly budget and track remaining balance
- Visual charts — Pie chart by category, Bar chart by month
- Filter expenses by month
- Profile page with spending summary
- Responsive UI with Bootstrap 5

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Python, Flask |
| Database | SQLite |
| Frontend | HTML, CSS, Bootstrap 5 |
| Charts | Chart.js |
| Auth | Werkzeug (password hashing) |

## Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/Hitha-04/Expense-Tracker.git
cd Expense-Tracker
```

### 2. Create and activate virtual environment
```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Run the app
```bash
python app.py
```

### 5. Open in browser
